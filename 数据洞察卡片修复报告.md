# 数据洞察卡片动态数据修复报告

> 修复概览页面数据洞察卡片的百分比圆环展示问题

---

## 🔧 修复的问题

### 问题描述
数据洞察的 4 个卡片的百分比圆环显示固定数值，不会根据后端返回的真实数据动态变化：
- 项目卡片：固定 74%
- 测试用例卡片：固定 82%
- 缺陷卡片：固定 44%
- 执行次数卡片：固定 91%

---

## ✅ 修复方案

### 修改文件
**文件：** `frontend/src/Dashboard/Overview.vue`

### 修改内容

**修复前：**
```javascript
const insightCards = computed(() => [
  {
    key: 'projects',
    progress: 74  // ❌ 固定值
  },
  {
    key: 'testcases',
    progress: 82  // ❌ 固定值
  },
  {
    key: 'bugs',
    progress: 44  // ❌ 固定值
  },
  {
    key: 'executions',
    progress: 91  // ❌ 固定值
  }
])
```

**修复后：**
```javascript
const insightCards = computed(() => {
  // 计算每个指标的百分比（基于总数）
  const total = totalAssets.value || 1
  return [
    {
      key: 'projects',
      progress: Math.round((metrics.value.total_projects / total) * 100) || 0  // ✅ 动态计算
    },
    {
      key: 'testcases',
      progress: Math.round((metrics.value.total_testcases / total) * 100) || 0  // ✅ 动态计算
    },
    {
      key: 'bugs',
      progress: Math.round((metrics.value.total_bugs / total) * 100) || 0  // ✅ 动态计算
    },
    {
      key: 'executions',
      progress: Math.round((metrics.value.total_executions / total) * 100) || 0  // ✅ 动态计算
    }
  ]
})
```

---

## 📊 计算逻辑

### 百分比计算公式
```
卡片百分比 = (该项数量 / 总资产数) × 100%
```

其中：
- **总资产数** = 项目数 + 用例数 + 缺陷数 + 执行次数
- **该项数量** = 当前卡片对应的指标值
- **容错处理**：当总数为 0 时，使用 1 作为除数，避免除零错误
- **结果处理**：使用 `Math.round()` 四舍五入取整，当结果为 0 时显示 0

## 🎨 技术细节

### 1. 响应式更新
- 使用 `computed` 计算属性，当 `metrics` 或 `totalAssets` 变化时自动重新计算
- Vue 的响应式系统确保视图自动更新

### 2. 数据绑定
```vue
<svg viewBox="0 0 36 36" aria-hidden="true">
  <path
    d="M18 2.0845 a 15.9155 15.9155 0 0 1 0 31.831 a 15.9155 15.9155 0 0 1 0 -31.831"
    fill="none"
    stroke="rgba(255,255,255,0.18)"
    stroke-width="3"
  />
  <path
    d="M18 2.0845 a 15.9155 15.9155 0 0 1 0 31.831 a 15.9155 15.9155 0 0 1 0 -31.831"
    fill="none"
    stroke="white"
    stroke-width="3"
    :stroke-dasharray="`${card.progress}, 100`"  <!-- 动态绑定百分比 -->
  />
</svg>
```

### 3. 圆环原理
- 使用 SVG 的 `stroke-dasharray` 属性控制圆环显示
- `stroke-dasharray="74, 100"` 表示显示 74% 的圆弧
- 第一个值是显示的长度，第二个值是总长度

## 📝 注意事项

1. **数据源**：数据来自 `metricsAPI.getOverview()` 接口
2. **初始状态**：当后端数据为空时，所有圆环显示 0%
3. **比例关系**：4 个卡片的百分比总和约为 100%（四舍五入可能有微小误差）
4. **响应式**：数据变化时，圆环会自动更新，无需手动刷新
