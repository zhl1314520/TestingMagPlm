# 三、数据库设计文档（Database Design）

## 1️⃣ 用户表

```
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    role VARCHAR(20),
    email VARCHAR(100),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

------

## 2️⃣ 项目表

```
CREATE TABLE projects (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100),
    description TEXT,
    owner_id INT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

------

## 3️⃣ 项目成员表

```
CREATE TABLE project_members (
    id INT PRIMARY KEY AUTO_INCREMENT,
    project_id INT,
    user_id INT,
    role VARCHAR(20)
);
```

------

## 4️⃣ 测试用例表

```
CREATE TABLE test_cases (
    id INT PRIMARY KEY AUTO_INCREMENT,
    project_id INT,
    module VARCHAR(100),
    title VARCHAR(255),
    steps TEXT,
    expected TEXT,
    status VARCHAR(20),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

------

## 5️⃣ 执行记录表

```
CREATE TABLE executions (
    id INT PRIMARY KEY AUTO_INCREMENT,
    project_id INT,
    type VARCHAR(20),
    status VARCHAR(20),
    result TEXT,
    created_at DATETIME
);
```

------

## 6️⃣ 缺陷表

```
CREATE TABLE bugs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    project_id INT,
    testcase_id INT,
    title VARCHAR(255),
    description TEXT,
    status VARCHAR(20),
    priority VARCHAR(20),
    assignee_id INT,
    created_at DATETIME
);
```

------

## 7️⃣ 报告表

```
CREATE TABLE reports (
    id INT PRIMARY KEY AUTO_INCREMENT,
    project_id INT,
    pass_rate FLOAT,
    fail_rate FLOAT,
    created_at DATETIME
);
```

------

## 8️⃣ 日志表

```
CREATE TABLE logs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,
    action VARCHAR(255),
    detail TEXT,
    created_at DATETIME
);
```
