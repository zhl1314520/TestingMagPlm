## 📊 数据库字段映射

### users 表
```sql
id, username(15), password(110), role, email(50), 
created_at, updated_at, deleted_at
```

### projects 表
```sql
id, name(100), description, owner_id, 
created_at, updated_at, deleted_at
```

### project_members 表
```sql
id, project_id, user_id, role, joined_at
```

### test_cases 表
```sql
id, project_id, module(100), title(255), steps, expected,
status, priority, created_by,
created_at, updated_at, deleted_at
```

### executions 表
```sql
id, project_id, name(200), type, status, result,
total_cases, passed_cases, failed_cases, pass_rate, executed_by,
created_at, updated_at
```

### bugs 表
```sql
id, project_id, testcase_id, title(255), description,
status, priority, assignee_id, reporter_id,
created_at, updated_at, deleted_at
```

### reports 表
```sql
id, project_id, execution_id, title(200),
pass_rate, fail_rate,
total_cases, passed_cases, failed_cases, created_by,
created_at, updated_at
```

## 📝 使用示例

### 创建用户
```python
from schemas.user import UserCreate
from DAO import user as user_dao

user_data = UserCreate(
    username="new_user",
    email="user@example.com",
    role="tester",
    password="hashed_password"
)
new_user = await user_dao.create_user(db, user_data)
```

### 查询项目用例
```python
from DAO import testcase as testcase_dao

testcases = await testcase_dao.get_testcases_by_project(
    db, 
    project_id=1,
    skip=0,
    limit=20
)
```

### 更新执行状态
```python
from DAO import execution as execution_dao

execution = await execution_dao.update_execution_status(
    db,
    execution_id=1,
    status="已完成",
    total_cases=10,
    passed_cases=8,
    failed_cases=2
)
# 自动计算 pass_rate = 80.00
```

### 创建缺陷
```python
from schemas.bug import BugCreate
from DAO import bug as bug_dao

bug_data = BugCreate(
    project_id=1,
    title="登录页面偶发性白屏",
    description="详细描述...",
    status="新建",
    priority="p1",
    assignee_id=2,
    reporter_id=1,
    testcase_id=1
)
new_bug = await bug_dao.create_bug(db, bug_data)
```
